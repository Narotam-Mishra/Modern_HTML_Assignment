Question 2 - What are HTML Entities? List out 5 commonly used HTML entities.

# HTML Entities - An HTML entities are a piece of text ("string") that begins with an ampersand (&) and ends with a semicolon (;). Entities are frequently used to display reserved characters which would otherwise be interpreted as HTML code, and invisible characters like non-breaking spaces. We can also use them in place of other characters that are difficult to type with a standard keyboard. Character entities are used to display reserved characters in HTML. 

# Syntax for HTML entities

&entity_name;
or
&#entity_number;

# Commonly used HTML entities - 

1). non-breaking space - A non-breaking space is a space that will not break into a new line. Two words separated by a non-breaking space will stick together not break into a new line. This is handy when breaking the words might be disruptive. It is written as &nbsp;

2). copyright - It is written as &copy;

3). less than - It is written as &lt;

4). greater than - It is written as &gt;

5). ampersand - It is written as &amp;

6). double quotation mark - It is written as &quot;
